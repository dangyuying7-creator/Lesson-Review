一、快速启动
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. 双击 index.html，用 Chrome / Edge / Firefox 打开即可使用。
2. 首次打开需要联网加载 3 个 CDN 库（Tailwind CSS、Chart.js、SheetJS）。
   加载完成后，浏览器会自动缓存，之后无需联网即可离线使用。

如需完全离线运行，请参考「五、离线部署」章节。

三、使用流程
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Step 1  填写顶部信息栏
        老师姓名（必填）、科目、时长、开课时段

Step 2  上传课堂评论数据（可选但建议上传）
        · 支持 CSV / Excel（.xlsx / .xls）
        · 必须包含"评论内容"相关列（系统自动识别以下列名）：
          评论内容 / 内容 / content / 发言内容 / 评论 / 消息内容
        · 上传后自动预览前 10 行数据，可核对字段识别是否正确

Step 3  上传出勤转化数据 或 手动填写数值（至少填 5min/30min 转化率）
        · 上传后系统自动识别字段并回填到手动输入框
        · 自动识别列名包括：
          5分钟出勤转化率 / 5min转化率 / 30分钟出勤转化率 / 30min转化率 等
        · 转化率可填百分数（如 75）或小数（如 0.75，系统自动换算）

Step 4  点击右上角「▶ 生成评价报告」按钮
        · 自动计算综合评分（100分制）
        · 自动分析评论好评/差评
        · 生成优缺点清单和改进建议
        · 渲染图表

Step 5  查看结果 / 导出
        · 点击「📋 复制」复制完整报告文本
        · 点击「⬇️ 导出TXT」下载报告文件

四、打分规则说明
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

总分 100 分，分为两大维度：

【留存转化分 60分】
  5分钟转化率（25分）：
    ≥85%  →  25分
    75~84% →  22分
    60~74% →  18分
    45~59% →  13分
    30~44% →   8分
    <30%   →   3分

  30分钟转化率（35分）：
    ≥60%  →  35分
    45~59% →  29分
    30~44% →  22分
    20~29% →  14分
    10~19% →   7分
    <10%   →   2分

【互动评论分 40分】
  有效评论数量（15分）：
    ≥200条 → 15分  ≥100条 → 12分  ≥50条 → 8分
    ≥20条  →  5分  ≥10条  →  2分  <10条  → 1分

  正面评论占比（25分）：
    ≥80%  → 25分   ≥65% → 20分   ≥50% → 14分
    ≥35%  →  8分   ≥20% →  4分   <20%  →  1分

【评级划分】
  S 级（优秀）：90~100分
  A 级（良好）：80~89分
  B 级（合格）：70~79分
  C 级（待改进）：0~69分

五、自定义配置（修改 js/config.js）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

所有关键词、打分权重、评级标准均集中在 js/config.js 文件中，
用记事本即可编辑，修改后刷新浏览器生效。

常见修改场景：
  ▸ 添加好评词  →  修改 CONFIG.KEYWORDS.positive 数组
  ▸ 添加差评词  →  修改 CONFIG.KEYWORDS.negative 数组
  ▸ 调整5分钟分值门槛  →  修改 CONFIG.SCORING.fiveMin.tiers
  ▸ 调整30分钟分值门槛 →  修改 CONFIG.SCORING.thirtyMin.tiers
  ▸ 调整评级分数线     →  修改 CONFIG.RATINGS 数组

七、离线部署（完全不需要联网）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

下载以下 3 个文件放入 lesson-eval/ 目录，并修改 index.html 中的引用路径：

1. Tailwind CSS
   https://cdn.tailwindcss.com  → 保存为 libs/tailwind.min.js
   修改 index.html 第6行为：<script src="libs/tailwind.min.js"></script>

2. Chart.js
   https://cdn.jsdelivr.net/npm/chart.js@4.4.0/dist/chart.umd.min.js
   → 保存为 libs/chart.min.js
   修改 index.html 第7行为：<script src="libs/chart.min.js"></script>

3. SheetJS (xlsx)
   https://cdn.sheetjs.com/xlsx-0.20.2/package/dist/xlsx.full.min.js
   → 保存为 libs/xlsx.full.min.js
   修改 index.html 第8行为：<script src="libs/xlsx.full.min.js"></script>
